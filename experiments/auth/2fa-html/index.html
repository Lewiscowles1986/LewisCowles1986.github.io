This is an example of a browser-based nearly declarative OTP sign-in via the web. The number sending the message cannot be in your contacts. This sort of makes sense. You'll also need a HTTPS domain you are in control of. No server-side logic is needed, but there is a JavaScript snippet that I am using to have Chrome and browser-API vendors support the declarative format provided without JS in Safari.
